# 🎨 Background Color Changer

A simple JavaScript project that changes the background color of the webpage every second when you click **Start**, and stops when you click **Stop**.

---

## 🚀 Features

- Start changing background color every 1 second
- Stop the color changing anytime
- Generates random HEX colors
- Smooth UI with simple design

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)
